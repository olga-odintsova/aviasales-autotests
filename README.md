# Проект по автоматизации тестирования (UI & API)

## Описание проекта
Данный репозиторий содержит автоматизированные тесты для проверки веб-интерфейсов и API. Проект построен на базе языка Python с использованием фреймворка Pytest и паттерна проектирования Page Object Model (POM) для UI-тестов. Отчетность по результатам тестирования генерируется с помощью Allure.

🔗 **Финальный проект по ручному тестированию:** [https://olgaodintsova.yonote.ru/share/db4fa1b5-0949-4180-b5c9-a013040c1ed6](https://olgaodintsova.yonote.ru/share/db4fa1b5-0949-4180-b5c9-a013040c1ed6)

## Стек технологий
* **Язык:** Python 3
* **Фреймворк:** Pytest
* **UI Тестирование:** Selenium WebDriver, Webdriver Manager
* **API Тестирование:** Requests
* **Отчетность:** Allure

## Установка и настройка

1. **Склонируйте репозиторий и перейдите в папку проекта:**
```bash
git clone <ссылка_на_ваш_репозиторий>
cd <название_папки>
```

2. **Создайте и активируйте виртуальное окружение:**
* **Windows:**
```cmd
python -m venv venv
venv\Scripts\activate
```


* **macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```



## Инструкции по запуску тестов

В проекте настроены маркеры для раздельного запуска различных видов тестирования. Убедитесь, что вы находитесь в корневой директории проекта перед запуском.

* **Запуск только UI-тестов (Selenium):**
```bash
pytest -m "ui"
```


* **Запуск только API-тестов (Requests):**
```bash
pytest -m "api"
```


* **Запуск всех тестов в проекте:**
```bash
pytest
```


## Генерация Allure отчета

Для формирования наглядного отчета о прохождении тестов выполните следующие шаги:

1. **Запустите тесты с флагом сохранения результатов:**
```bash
pytest --alluredir=allure-results
```


2. **Сгенерируйте и откройте отчет в браузере:**
*(Требуется установленный локально Allure Commandline)*
```bash
allure serve allure-results
```
